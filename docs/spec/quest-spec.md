# 퀘스트 시스템 기획

## 구조

퀘스트는 **장기(목표)** 와 **단기(루틴)** 두 레벨로 나뉜다.

| 구분 | 설명 | 예시 |
|---|---|---|
| 장기 | 큰 목표. 직접 완료 체크하지 않고 하위 단기가 모두 완료되면 달성 | 자격증 따기 |
| 단기 | 장기를 위한 작은 실행 단위. 체크하면 완료 탭으로 이동 | 따뜻한 물로 손 씻기 |

- 단기는 `parentId`로 장기에 연결될 수 있음 (연결 없이 독립 단기도 가능)

---

## 탭

| 탭 | 내용 |
|---|---|
| 단기 | 단기 퀘스트 목록. 미완료가 위, 완료가 아래 (취소선+흐림). 장기에 연결된 항목은 카드 상단에 장기 제목 라벨 표시 |
| 장기 | 장기 퀘스트 목록. 미완료가 위, 완료가 아래. 하위 단기 진행률 프로그레스 바 (`완료수/전체`) 표시 |

- 완료 탭 없음 — 완료된 항목은 각 탭 내에서 아래로 정렬되어 표시

---

## 퀘스트 추가

### 단기 탭에서 "+" 클릭
- 모달 없이 리스트에 빈 카드 추가 → 인라인 입력
- 포인트 고정 10

### 장기 탭에서 "+" 클릭
- 루틴 모달 오픈 (장기 퀘스트 새로 추가)

### 장기 카드의 "오늘 이 목표 위해 작은 걸 적어볼까요?" 클릭
- 루틴 모달 오픈
- 기존 단기(미완료 + 완료) 목록 표시 (완료는 취소선)
- "+" 버튼으로 새 루틴 입력란 추가
- "다 했어요" → 새 입력만 단기 탭에 추가 (`parentId` = 해당 장기 ID, 포인트 20)

---

## 퀘스트 완료

### 단기 체크 시
- 단기 탭에서 빠지고 완료 탭으로 이동
- 체크 원 bounce 애니메이션만 (토스트 없음)

### 장기 100% 달성 시 (연결된 단기 모두 완료)
- CelebrationToast: "목표 달성! 🎊" / "'{장기제목}' 완료 +{포인트} XP"

---

## 퀘스트 관리 (⋮ 메뉴)

### 전 탭 공통
- **수정**: 제목을 인라인 input으로 전환
- **삭제**: 즉시 제거

### 완료 탭 전용
- **돌려놓기**:
  - `parentId` 없음 → 바로 단기로 복원 (모달 없음)
  - `parentId` 있음 → 확인 모달:
    - **돌려놓기**: 단기로 복원 + 해당 장기가 완료 상태였으면 장기도 함께 복원
    - **추가하기**: 완료 기록 유지 + 동일 제목의 새 단기 생성 (parentId 없음, 10포인트)
    - **취소**: 모달 닫기

---

## 포인트 체계

| 항목 | 포인트 |
|---|---|
| 단기 (인라인 추가) | 10 |
| 단기 (장기 모달에서 추가) | 20 |
| 장기 | 100 (직접 설정 가능 예정) |
