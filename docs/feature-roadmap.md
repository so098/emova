# Feature Roadmap

## 데이터 시각화

현재 Supabase에 퀘스트 데이터가 쌓이고 있으므로, 이를 활용한 시각화 기능 추가 예정.

### 후보 기능

| 기능 | 내용 | 난이도 | 필요 데이터 |
|---|---|---|---|
| 주간 완료 차트 | 최근 7일 퀘스트 완료 수 막대/라인 그래프 | 낮음 | quests (completed_at) |
| XP 성장 곡선 | 누적 XP를 시간축으로 보여주는 라인 차트 | 중간 | xp_ledger |
| 달성률 캘린더 | GitHub 잔디처럼 일별 달성률을 색상으로 표시 | 중간 | quests (completed_at) |
| 감정 히스토리 | 감정 선택 기록을 타임라인/히트맵으로 표시 | 높음 | survey_emotions (세션 연동 필요) |

### 차트 라이브러리 후보
- **Recharts** — React 친화적, 가벼움, 커스터마이징 용이
- **순수 SVG** — 의존성 없음, 완전한 디자인 자유도
